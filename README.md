# ⚔️ Vinland Saga Projeto

---

## Sobre o Projeto

Este projeto é uma aplicação web desenvolvida como parte de um exercício acadêmico, utilizando o tema **Vinland Saga** para criar uma experiência interativa com páginas de autenticação e simulação.

---

## 📄 Páginas do Sistema

- 🏠 Home (index) – Apresentação do site e personagens  
- 🔐 Login – Autenticação de usuários  
- 📝 Cadastro – Registro com código de ativação  
- 📊 Sobre Mim – Área falando sobre o desenvolvedor do codigo Guilherme Barbosa De Albuquerque

---

## ⚙️ Funcionalidades

- Cadastro de usuários com validação  
- Login com autenticação via API  
- Uso de código de ativação para vincular empresa  
- em desenvolvimento (Gerenciamento de sessão do usuário)

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Node.js (backend)  
- MySQL  

---

## 🔄 Fluxo do Sistema

1. Usuário acessa o site  
2. Realiza cadastro com código de ativação  
3. Faz login com e-mail e senha  
4. É redirecionado para áreas internas do sistema
